Timestamp human readable filter
===
# What's this?
Convert unixtimestamp/epoch in sentence (json , csv , plaintext ...) to human readable.

# Install
python3 (no any pip)

# Run
`$ tsfilter < test.data`

# Specs
## max/min
@ 10 digits
min: 1300000000 = 2011-03-13T16:06:40
max: 1999999999 = 2033-05-18T12:33:19

## supported types
see `test.data`

hoge
fuga
